# Expense Tracker

A simple desktop Expense Tracker application built using Python and Tkinter. This project helps users record daily expenses, view total spending, and save data in a CSV file.

## Features

- Add expense details
- Enter date, description, amount, and category
- View total expense
- Save expenses to CSV file
- Simple and user-friendly GUI
- Data persists after restarting the app

## Technologies Used

- Python
- Tkinter
- CSV

## Project Structure

expense-tracker/
│── expenseTracker.py  
│── expenses.csv  
│── README.md  

## How to Run

1. Download or clone the project
2. Open the project folder in VS Code
3. Open `expenseTracker.py`
4. Right-click inside the file and select **Run Python File**

## Requirements

- Python 3.x (installed via Python or Anaconda)
- Tkinter (built-in with Python)
- VS Code (optional)
